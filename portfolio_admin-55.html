📋 How It Works
  1. Edit portfolio_funds.csv in Excel or Google Sheets
  2. To change preset defaults, update the aggressive, balanced, conservative columns on the first row of each sub-asset
  3. Drop the CSV here → click Preview to verify structure, funds, and preset weights
  4. Click Generate & Download
  5. The downloaded portfolio_tool.html has your fund data and preset weights baked in
  6. Rename to index.html and push to GitHub Pages
  7. This admin file and the CSV stay private
